Introducere în Git și GitHub
Git este un instrument (software) care rulează local pe calculatorul tău. Este ca o „mașină a timpului” pentru codul tău: salvează versiuni ale proiectului, permițându-ți să revii la o stare anterioară dacă ceva se strică.
GitHub este o platformă online (cloud) unde îți găzduiești proiectele create cu Git. Este ca o rețea socială pentru programatori, unde poți colabora cu alții, poți arăta portofoliul tău și poți păstra codul în siguranță.
 Obiective:
1. Gestionarea locală presupune inițierea proiectului cu git init și organizarea modificărilor prin adăugarea lor în Staging Area cu git add, urmată de salvarea lor permanentă într-un commit.
2. Interacțiunea cu GitHub îți permite să descarci proiecte existente pe calculatorul tău prin git clone și să îți publici propriile versiuni de cod pe serverele online folosind comanda git push.
3. Lucrul pe ramuri (branching) oferă libertatea de a experimenta idei noi într-un spațiu izolat, asigurându-te că versiunea principală a proiectului rămâne stabilă și funcțională în timp ce tu dezvolți funcții noi.
